# Software para uso de un vivero

<p>El objetivo del desarrollo de este software es para poder llevar a cabo el uso y gestión de un vivero en Coyoacán (Ciudad de México), en el cuál se manejan distintos procesos como 
lo es la compra de productos para su venta, venta de ejemplares, manetenimiento de los mismos, registro de los productos que nos brindan los proveedores, nomina de nuestros trabajadores;
así como lo es una vista para usuario que permita compras y demás movimientos.</p>

## Tecnologías usadas: 
-  JavaScript como lenguaje principal
-  NodeJs como framework para controlar el backend junto con express para el uso de protocolos http
-  PostgesSQL para gestionar la base de datos relacional
