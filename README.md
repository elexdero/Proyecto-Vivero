# Software para gestión de un vivero
<p>Este proyecto esta enfocado en crear un software para la gestión de un vivero, en el cual, nos encargamos de realizar un desarollo de la base 
de datos del vivero tomando en cuenta los apartados como puede ser un registro de los trabajadores, credenciales para poder acceder al software, información de las plantas,
productos, las ventas generadas, entre otros aspectos que se pudieron definir.
</p>

## Tecnologías usadas: 

  - JavaScript
  - NodeJs para manejar un backend más solido con js
  - SQL (se esta utilizando una base de datos relacional)
